# Rice Quality System

A full-stack application to analyze rice quality using AI (YOLO), Node.js, and Angular.

## Project Structure

-   `ai/`: Python environment for YOLOv8 model training and inference.
-   `backend/`: Node.js (Express) API server.
-   `frontend/`: Angular web application for users.
-   `database/`: Database scripts and configuration (PostgreSQL).

## Prerequisites

-   Node.js (v18+)
-   Python (v3.9+)
-   PostgreSQL (v14+)
-   Angular CLI (`npm install -g @angular/cli`)
